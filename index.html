代码美化

效果如下

.prettyprint *{font-family:'courier new',monospace;}
.prettyprint .com { color: #93a1a1; }
.prettyprint .lit { color: #AE81FF; }

如何做：
1、引入下面的js和css 

<link rel="stylesheet" type="text/css" href="http://cdn.bootcss.com/prettify/r298/prettify.min.css">
<link rel="stylesheet" type="text/css" href="css/prettify.override.css">
<script type="text/javascript" src='http://cdn.bootcss.com/prettify/r298/prettify.min.js'></script>
<script type="text/javascript" src='js/prettify.load.js'></script>

2、将你的代码放到<pre>这里是代码</pre>中。
这样就OK了。

说明一下

代码美化使用的是Google的prettify，地址：https://github.com/google/code-prettify
其中的[css/prettify.override.css]和[js/prettify.load.js]是我自己的css和js文件。
[css/prettify.override.css]是对[prettify.min.css]的重写，我修改了一下样式。
而[js/prettify.load.js]里面的代码如下： 

window.onload = function(){
	var pres = document.getElementsByTagName('pre');
	for(var i=0; i < pres.length;i++){
		var item = pres[i];
		item.setAttribute('class','prettyprint');
	}
	prettyPrint();
};

剥去我的东西，其实就是引入下面两个 

<link rel="stylesheet" type="text/css" href="http://cdn.bootcss.com/prettify/r298/prettify.min.css">
<script type="text/javascript" src='http://cdn.bootcss.com/prettify/r298/prettify.min.js'></script>
然后在需要美化的地方加上prettyprint类，再调用prettyPrint()方法。
